# Project Constraints

- **Browser Usage**: DO NOT open a browser or use the `browser_subagent` for testing or verification unless explicitly requested by the USER. All verification should be done via terminal output or file content checks.
- **Local Server Port**: The Revelti project must ALWAYS be launched on port 3001 (`npx live-server --port=3001`), so it won't conflict with another project.
