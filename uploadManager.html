Folder Setup

Folders are where your media is stored. You can later organize them, attach them to events, or turn them into galleries. Their structure can also define how galleries are organized. The first 3 levels can be used as filters (e.g. Year → Day → Category).

If left blank, the folder will be created in the root of the Studio Workspace.

Uploading to an existing folder will inherit its event association, collaborators, and gallery settings.
Attaching a folder to an event gives the event organizer access to view and manage this media.
Collaborators can view or manage the folder depending on the permissions you grant.
